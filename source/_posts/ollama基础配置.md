官网：https://ollama.com/
ollama版本：ollama version is 0.6.2

安装的方式有两种，一种是docker安装，一种是脚本安装。ollama是跨平台的，可根据自己系统进行选择，这里以Linux系统为例，同时选择一键脚本，安装命令如下：
```bash
curl -fsSL https://ollama.com/install.sh | sh
```
### 查看安装后的ollama在哪里
```bash
whereis ollama
```
